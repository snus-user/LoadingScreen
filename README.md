# Tilt Loading Page

Статическая страница загрузки на чистых HTML/CSS/JS.

## Как запустить локально

- Открой `index.html` в браузере.

Если нужен локальный сервер (чтобы всё работало как на хостинге):

- VS Code: Live Server
- Или любой простой static-server.

## Параметры URL

- `?ms=2600` — длительность «загрузки» в миллисекундах (900..12000)
- `?to=/app.html` — редирект после 100%

Пример:

- `index.html?ms=2200&to=https://example.com`
